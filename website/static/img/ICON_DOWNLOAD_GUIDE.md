# 特性图标下载指南

主页的5个特性图标需要从 undraw.co 下载对应的SVG文件。

## 需要下载的图标

1. **极简开发** (`undraw_workflow.svg`)
   - 搜索关键词：workflow, automation, development
   - 推荐链接：https://undraw.co/illustrations (搜索 "workflow" 或 "automation")
   - 下载后保存为：`website/static/img/undraw_workflow.svg`

2. **高成功率** (`undraw_success.svg`)
   - 搜索关键词：success, achievement, target
   - 推荐链接：https://undraw.co/illustrations (搜索 "success" 或 "achievement")
   - 下载后保存为：`website/static/img/undraw_success.svg`

3. **高性能** (`undraw_performance.svg`)
   - 搜索关键词：performance, speed, computing
   - 推荐链接：https://undraw.co/illustrations (搜索 "performance" 或 "speed")
   - 下载后保存为：`website/static/img/undraw_performance.svg`

4. **活跃的社区** (`undraw_community.svg`)
   - 搜索关键词：community, collaboration, team
   - 推荐链接：https://undraw.co/illustrations (搜索 "community" 或 "collaboration")
   - 下载后保存为：`website/static/img/undraw_community.svg`

5. **安全可靠** (`undraw_security.svg`)
   - 搜索关键词：security, protection, shield
   - 推荐链接：https://undraw.co/illustrations (搜索 "security" 或 "protection")
   - 下载后保存为：`website/static/img/undraw_security.svg`

## 下载步骤

1. 访问 https://undraw.co/illustrations
2. 在搜索框中输入对应的关键词
3. 选择合适的图标（建议选择风格一致的图标）
4. 点击图标，选择 "Download SVG"
5. 将下载的SVG文件重命名并保存到 `website/static/img/` 目录

## 注意事项

- 所有图标都应该是SVG格式
- 建议选择风格统一的图标系列
- 可以调整SVG的颜色以匹配网站主题色
